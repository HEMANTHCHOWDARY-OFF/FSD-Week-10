# Language Learning Flashcards (V2)

A clean, modern, and fully interactive flashcard application built with Vanilla HTML, CSS, and JavaScript. This project is designed for language learners who want a simple yet powerful tool to practice vocabulary with 3D flip animations and progress tracking.

## 🚀 Key Features

- **3D Flip Animations**: Realistic card-flipping experience.
- **Progress Tracking**: Real-time stats for "Studied" cards and percentage completion.
- **Local Storage**: Your cards and progress stay saved even after refreshing the page.
- **Full Customization**: Add your own cards, categories, and phrases.
- **Keyboard Shortcuts**: Spacebar to flip, Arrow keys to navigate.
- **Mobile Responsive**: Works perfectly on desktops, tablets, and smartphones.

## 📂 File Structure

- `index.html`: The core structure and modal system.
- `style.css`: Modern styling with glassmorphism and 3D transforms.
- `script.js`: The "brain" of the app using Object-Oriented Programming (OOP).
- `DOCUMENTATION.md`: A detailed beginner-friendly guide to the entire codebase.

## 🛠️ How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start learning!
